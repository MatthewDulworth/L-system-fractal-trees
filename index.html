L-System Fractal Tree

This sketch uses an L-system to generate a sequence of instructions which are then parsed using turtle grpahics to create a fractal tree.

The L-System starts with the axiom "F" and the uses the rule "F -> FF+[+F-F-F]-[-F+F+F]" to generate instructions for the turtle.

The turtle grpahics system interprets each instruction as follows:

  • "F" means draw a line
  • "+"" means rotate right
  • "-'' means rotate left
  • "[" means save the current location and translation data
  • "]" means revert to the previously save
